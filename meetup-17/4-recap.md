# Recap

## Summary

Given a string return the an array of all the possible permutations of capital and lowercase letters on that string.

## Input

A string.

## Output

An array of strings.

## Example

Input: `cat`

Output: `['cat', 'Cat', 'cAt', 'caT','CAt','CaT','cAT','CAT']`

## Constraints

Cannot use loops. Can only use recursion.
